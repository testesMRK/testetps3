# Projeto Template do GitHub

Este é um projeto base projetado para ser usado como um **GitHub Template**. Ele contém uma estrutura organizada para facilitar o início de novos desenvolvimentos.

## ✨ Características

- **Estrutura Organizada**: Pastas claras para código, arquivos públicos e mensagens.
- **Internacionalização**: Suporte básico para multi-idiomas na pasta `messages/`.
- **Configurações do VS Code**: Regras de formatação e extensões recomendadas prontas.
- **GitHub Actions**: Workflow de CI básico configurado.
- **Issue Templates**: Modelos prontos para Bug Reports e Feature Requests.

## 📁 Estrutura de Pastas

- `.github/`: Configurações do repositório, workflows e templates de issue.
- `.vscode/`: Configurações específicas para o Visual Studio Code.
- `messages/`: Arquivos JSON para traduções (pt-BR, en).
- `public/`: Arquivos estáticos (HTML, imagens, etc.).
- `src/`: Código-fonte principal do projeto.

## 🚀 Como usar este template

1. Clique no botão **"Use this template"** no topo do repositório no GitHub.
2. Dê um nome ao seu novo repositório.
3. Clone o seu novo repositório e comece a codar!
